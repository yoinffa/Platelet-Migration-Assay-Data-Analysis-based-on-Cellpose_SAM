# Platelet-Migration-Assay-Data-Analysis-based-on-Cellpose_SAM
Here are set of software and fiji plugin developed by Yongfa Dai, for Platelet Migration Assay experiment. Following step of introduction.png you can get the morphology and migration tracks data from platelet migration experiment.
